Francis#Http

Francis#Http

constructor

an http class to make requests over the net with retry and interval between them

constructor(requestParams: object, options: Any, callback: [function])
Parameters
requestParams (object) object where you can specify the options of the request
Name Description
requestParams.type [string] (default GET) the type of the request: possible values POST, GET, PUT, DELETE and JSONP
requestParams.url string the url to request for
requestParams.headers [object] (default {}) the headers object
requestParams.timeout [string] (default 2000) the timeout of the request in ms
requestParams.attempt [string] (default 1) attempts. if it fails for some reason retry
requestParams.retryAfter [number] (default 0) the interval between requests in ms: 500 for example
requestParams.async [boolean] (default true) the request could be synchrounous, default async
options (Any)
callback ([function] (default function(){}) ) onend callback
Example
<pre>
npm install --save http-francis

import { Http, getImageRaw, JSONPRequest } from 'http-francis';
OR
var Http = require("http-francis").Http;
OR
<script src='http-francis.js'></script> 
// in global case window['http-francis'].Http
 var getTask = new Http({
      method: "GET",
      url: "https://someimageurl/image.png",
      responseType: "blob",
      mimeType: "image/png",
      onProgress:(percentage)=>{ 
          // there must be Content-Length header in the response to get the right percentage
          // otherwise percentage is a NaN
      }
  });

  getTask.promise.then((response) => { 
      var imgTag = document.createElement("img");
      imgTag.src = response[0];
      document.body.appendChild(imgTag);       
  });
</pre>
